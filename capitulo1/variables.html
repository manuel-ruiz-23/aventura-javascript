Variables

Una variable es un nombre que hace referencia a un valor. Las variables se declaran usando 'var' seguido del nombre de la variable.

ejemplo:

var nombre;

La variable anterior esta declarada, pero aun no tiene ningun valor, podemos asignarle un valor de la siguiente manera:

var nombre = "Naruto";

Para continuar abre la consola una vez mas e ingresa:

var nombre = "tu_nombre";

puedes escribir 'nombre' para ver el valor guardado

una vez que hayas ingresado tu nombre:

continuar(nombre);
